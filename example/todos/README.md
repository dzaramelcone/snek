# Todo App — Full Showcase Example

This is the **full-feature showcase** for snek. It exercises the complete API
surface: DI, Redis, JWT, OAuth, WebSocket, SSE, `app.gather`, OpenAPI docs,
middleware chains, and database transactions.

This example validates the **final integration milestone** (Phase 16), not an
early-phase target. For simpler starting points see:

- `example/hello/` — Phase 9 milestone (routing + JSON, no dependencies)
- `example/db_basic/` — Phase 13 milestone (routing + one DB query)
